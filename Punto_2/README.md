# Codigo misterioso
_explicacion breve de la logica del programa_
```
El programa comienza enviando por referencia un numero de tipo entero a una funcion de tipo void donde se le realiza un proceso de modificacion.
En la primera funcion lo que hace el programa es invertir el numero de manera que ingresa 452 y sale como 254. La funcion trabaja recibiendo por referencia el parametro y dentro de ella se lo invierte usando variables auxiliares y un ciclo while (estructura iterativa), por el cual nombre a la funcion como "invertir_numero".

La segunda funcion solo saca la mitad del numero, por el cual le agregue el nombre de "extraer_mitad".

La tercera funcion solo va quitando digitos y los va guardando en una variable auxiliar, luego la suma a la variable original, por eso le puse de nombre "sumar_digito".
```
